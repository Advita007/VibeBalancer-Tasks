#### ***What are branches and why are they useful?***

________________

  * Branches are categorisations of commits that are made for different purposes, like creating different features, fixing a bug, etc.
  * They are used by users and developers to work on a part of the project, commit their changes to it and create a pull request for it to be included in the main branch of the project.
  * Users can work inside these branches, and experiment well as none of these would be reflected in their main branch unless the pull request is accepted. 
  * Branches allow its users to work on multiple aspects of a project at once and also helps the owners to check on various of these distinctly, as all of them would have had a different pull request associated with them.
