#### ***Cloning a repository from GitHub and and making changes in it***

______

#### The following steps were followed: 
  * ###### Forking the online repository to get a personal copy of the project in which we can make the desired changes.
  * ###### Then, we change the change the working directory to Task1 and enter into the project. Inside this using the command ` mkdir ` we create a subfolder and using ` touch ` we add files in it.
  * ###### We add all our folders and files into the staging area ready to be commited using ` git add . `,post which we can commit all our changes. 
  * ###### Lastly to sinc it with our online rspository by using `git push origin master`, we're pushing the master branch since that was where the changes were made and git knows the url of the online repo since we had previously cloned it and it had been set as default then.
